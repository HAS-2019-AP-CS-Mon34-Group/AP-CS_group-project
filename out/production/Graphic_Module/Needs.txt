* Program Features

1) 회원가입 기능
- 아이디
- 패스워드
- 회원가입 시 자료를 보관할 파일을 가입한 아이디로 생성
- 아이디 중복 확인 기능

2) 대화패턴 분석
- 일자별 대화량 비교 (단어 개수 기준 / 말풍선 개수 기준)
- 사람 당 대화량 비교 (일자별)
- 시간대별 대화량 비교
> 대화량 비교는 최대 지점을 찾아내는 것으로 진행
- 이모티콘 빈도수 체크
- 반복 단어 체크

3) 파일 저장
- 로그인한 아이디로 분석 결과를 .txt 파일로 저장
- 추후 오픈 가능하도록 함 (파일 이름 생성 규칙 만들기)

4) 비회원 분석 기능
- 일회성 자료 제공 (콘솔 사용)
